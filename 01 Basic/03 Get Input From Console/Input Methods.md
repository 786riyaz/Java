| Scanner Method  | Data Type Read          | Example Input | Notes                                 |
| --------------- | ----------------------- | ------------- | ------------------------------------- |
| `nextInt()`     | int (whole number)      | 10            | Fails if input has decimals.          |
| `nextLong()`    | long                    | 12345678900   | Used for very large integers.         |
| `nextDouble()`  | double (decimal number) | 10.25         | Accepts decimals.                     |
| `nextFloat()`   | float (decimal number)  | 10.25         | Less precise than double.             |
| `nextByte()`    | byte                    | 120           | Range: -128 to 127.                   |
| `nextShort()`   | short                   | 25000         | Range: -32,768 to 32,767.             |
| `nextBoolean()` | boolean                 | true / false  | Case-insensitive.                     |
| `next()`        | String (single word)    | Hello         | Stops at space.                       |
| `nextLine()`    | String (full line)      | Hello world   | Reads complete line including spaces. |
