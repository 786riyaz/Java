## Java Escape Characters (with one-line meaning)

* ' – Inserts a single quote character.
* " – Inserts a double quote character.
* \ – Inserts a backslash character.
* \n – Inserts a new line.
* \t – Inserts a horizontal tab.
* \r – Inserts a carriage return (cursor moves to start of line).
* \b – Inserts a backspace (removes previous character in output).
* \f – Inserts a form feed (page break, rarely used).
* \uXXXX – Inserts a Unicode character using its hexadecimal code.
* \0 – Inserts a null character (rare in Java).

---

## Additional Notes

* Escape characters are used inside string literals and char literals.
* They always start with a backslash .
* They help represent characters that cannot be typed directly or have a special meaning.
